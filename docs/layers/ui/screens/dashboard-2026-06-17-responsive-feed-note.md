# 대시보드 화면 기준 보강

## 2026-06-17 변경

- 브라우저를 데스크톱 절반 폭 정도로 줄여도 핵심 지역별 상승률, 주요 부동산 지표, 콘텐츠 피드가 겹치거나 박스 밖으로 튀어나오지 않아야 한다.
- 모바일 폭까지 기다리지 않고 1200px 이하 중간 폭에서 먼저 1열/2열로 접히는 반응형 기준을 둔다.
- 1200px 이하에서는 전체 프레임과 상단/푸터의 데스크톱 최소 폭을 해제해 반쪽 화면에서 가로 스크롤로 밀리지 않게 한다.
- 1200px 이하에서는 오른쪽 고정 레일을 숨기고, 라이브 티커/반응 캐러셀처럼 화면 밖에 놓이는 요소가 문서 폭을 키우지 않도록 가로 overflow를 차단한다.
- 주요 부동산 지표 카드는 출처/기준시각 라벨이 값 영역 위에 겹치지 않도록 카드 내부 흐름에 포함하고, 긴 숫자/단위는 카드 폭 안에서 줄바꿈한다.
- 대시보드 콘텐츠 피드는 `all` feed를 한 번 가져와서 클라이언트에서 자르는 방식이 아니라 뉴스, 리포트, 영상, 블로그/커뮤니티별로 각각 조회한다.
- 각 콘텐츠 카테고리는 최대 5개 최신 후보를 보여준다.
- 백엔드에서 `contentType=link`, `blog`, `community`로 들어온 항목은 검색 후보여도 블로그/커뮤니티 피드에 남긴다.
- 최근 이슈 수집은 지역/단지 쟁점 키워드에 더해 `블로그`, `영상`, `리포트` 검색을 함께 수행하고, 검색 API에는 최근 1개월/날짜 우선 힌트를 붙인다.
- 최신 콘텐츠 후보는 검색 의도와 도메인을 맞춘다. `블로그` 쿼리는 블로그/카페류만, `영상` 쿼리는 영상 도메인만, `리포트` 쿼리는 리포트성 출처/문구만 남긴다.
- 검색 후보 중 전년도 발행 글, 90일 초과 프론트 후보, 45일 초과 수집 후보, 경매/공장/생활형 홍보 글은 최신 피드에서 제외한다.
- 대시보드 콘텐츠 조회는 카테고리별 100건까지 가져온 뒤 필터링해, 노이즈를 제거해도 실시간 뉴스가 최대 5개 채워지도록 한다.
- 영상 피드는 일반 매물/경매/홍보 유튜브를 제외하고, 부읽남TV, 집코노미, 매부리TV, 삼프로TV, 김작가TV, 땅집고, KB부동산TV, 부동산R114 등 분석/리서치 채널 신호가 있는 후보만 남긴다.
- 검증된 영상 후보가 없으면 매물 홍보형 영상을 대신 채우지 않고, 분석 채널 후보 없음 상태를 그대로 표시한다.
