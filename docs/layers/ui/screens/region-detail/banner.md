# 지역/단지 상세 요약 배너

## 목적

지역/단지 상세 상단에서 사용자가 가장 먼저 봐야 하는 한줄 브리핑과 근거 keyword를 보여주는 배너입니다. 자극적인 매수·매도 판단이 아니라, "왜 지금 이 지역을 봐야 하는가"를 짧고 분명하게 전달합니다.

## 표시 원칙

- 지역/단지 이름과 관찰 기간을 함께 보여줍니다.
- 가격 신호, 전세 신호, 공급 신호, 커뮤니티 반응을 분리합니다.
- 공식 지표와 커뮤니티 반응이 엇갈릴 때는 괴리 자체를 핵심 문장으로 둡니다.
- 데이터가 stale이면 배너 안에서 바로 드러냅니다.
- 개인화된 행동 지시는 쓰지 않습니다.

## 문구 후보

- "마포구는 전세수급이 먼저 달아오르고, 거래량은 아직 따라오지 않았습니다."
- "동탄역권은 GTX 기대가 언급량을 끌어올렸지만 입주 물량 부담이 같이 보입니다."
- "송도는 미분양 부담이 남아 있어 가격 반등보다 공급 소화가 먼저입니다."

## API 후보

| 필드 | 설명 |
| --- | --- |
| `headline` | 한줄 브리핑 |
| `subcopy` | 보조 설명 |
| `keywords` | 근거 keyword chip |
| `metricSignals` | 가격, 전세, 공급, 반응 신호 |
| `stale` | 데이터 지연 여부 |
| `personalizedSafe` | 개인화/행동 지시처럼 보일 위험 여부 |
