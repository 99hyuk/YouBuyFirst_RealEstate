# community

## 역할

커뮤니티 원천 데이터와 반응 분류를 담당합니다. 이 도메인은 좋은 입력을 안전하게 모으고, 글이 어떤 종목을 어떤 방향으로 말하는지 분석 가능한 형태로 넘깁니다.

## 담당 범위

- 커뮤니티/게시판 source registry
- 네이버 종토방, 에펨코리아, 뽐뿌, 디시 계열 등 수집 후보
- 공개 HTTP 수집과 Playwright fallback 기준
- 제한 원문 저장 정책
- 글/댓글의 종목 언급 후보
- `bullish`, `bearish`, `neutral` 반응 방향 분류
- 인기글/개념글/추천글/조회수 상위글을 통한 확산 확인

## 정본 문서

| 문서 | 역할 |
| --- | --- |
| `COLLECTION.md` | 수집 소스와 수집 정책 |
| `REACTION_GUIDE.md` | 커뮤니티 반응 용어와 표시 기준 |

## 다른 도메인과의 접점

- `stock`: 종목명, 티커, 별칭 매칭 기준을 참조합니다.
- `indicator`: 분류된 반응과 원천 통계를 30분 지표 입력으로 넘깁니다.
- `governance`: robots, 약관, 원문 재게시, 개인정보 리스크를 확인합니다.
- `layers/ui`: 원문 전문이 아니라 집계/대표 키워드/출처 링크 중심으로 노출합니다.

## 하지 않는 일

- 시세 provider 연동
- 개미 심리 지수 최종 산식 소유
- 모의 주문 체결
- AI 전략 판단
