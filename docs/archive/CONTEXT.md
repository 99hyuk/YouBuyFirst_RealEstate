# Archived Context Handoff

이 파일은 예전 문서 구조에서 쓰던 handoff입니다. 현재 새 채팅 시작 문서가 아니며, 정본으로 읽지 않습니다.

현재 기준:

- 전체 라우터: `AGENTS.md`
- 현재 상태: `docs/current/HANDOFF.md`
- 다음 작업: `docs/current/TASKS.md`
- 제품 방향: `docs/product/FINAL_PRODUCT_PLAN.md`
- 작업 영역 기준: `docs/layers/ops/WORK_AREAS.md`

예전 대화나 PR 맥락을 찾아야 할 때만 이 archive 폴더를 `rg`로 검색합니다. 오래된 `docs/workstreams/`, `docs/CURRENT_HANDOFF.md`, `docs/TASKS.md`, `crawl/data/front/trade` 같은 경로와 track 이름은 당시 기록으로만 봅니다.
